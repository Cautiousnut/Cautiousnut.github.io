1. Linux文件与目录管理

1.1. 一、路径

  • 根目录：/
  • 绝对路径：从根目录 / 写起，如/a/b/c
  • 相对路径：不是由根目录写起，比如从/a/b/c/e要到/a/b/c/d，则写为../d

1.2. 二、常用命令

  • ls 列出目录以及文件名
    • 语法：ls -[可选属性] 目录名称
    • 可选属性：
      • a： 将全部的文件连同隐藏的文件一同列出
      • d： 仅列出目录本身而不列出目录内的文件数据
      • l： 长数据串列出，包含文件的属性和权限等数据
      • 使用-al时则会列出所有的文件（包括隐含文件的属性和权限）

  • cd 切换目录
    • change directory
    • 语法格式： cd 相对路径或者绝对路径

      举例

      • 切换到d目录下
      • 切换到当前目录的上一级目录
  • pwd 显示当前所在目录
    • Print Working directory
    • 语法：pwd [-P]
    • 参数：-P 显示出确实的路径，而非使用连结路径

  • mkdir 创建新目录
    • make directory
    • 语法：
      • mkdir 目录名
        • 创建一个新的名为目录名的空文件
      • mkdir -m 权限值 已有目录绝对路径/新的文件目录名
        • 在一个已有目录中创建一个拥有相应权限值的空目录文件（mode）
        • 权限值的计算：r=4,w=2,x=1,每三位得到一个值，最终得到一个三位数
      • mkdir -p 绝对路径/新的目录名
        • 在绝对路径中创建一个空的目录文件。如果绝对路径下的目录存在，则直接创建，如果不存在，则创建这个目录，然后再创建新的目录文件(parents no error if existing, make parent directories as needed)
      • mkdir -v 目录名
        • 在当前位子创建一个空目录，创建完成后提示已创建目录（verbose print a message for each created directory）
      • mkdir 目录名1 目录名2 目录名3 ……
        • 在当前目录创建多个空目录文件
      • mkdir -vp 目录名1/目录名2
        • 在创建目录名1的基础上在这个目录里去创建目录名2这个空目录并在创建成功时提示信息
        • 是-v-p功能的叠加
      • mkdir -p -m 权限值 目录名称
        • 是-p -m的功能的叠加

  • rmdir（删除空的目录）
    • 语法：rmdir [-p] 目录名称
    • -p:连同上一级空的目录一同删除
    • 注意：只能删除空的目录

      实例：

  • cp（复制文件或者目录copy）
    • 语法：
      • 复制一个文件或者文件夹： cp [option] src target
      • 复制多个文件，目标是一个文件夹： cp [option] src1 src2 …… target
    • 常用选项：
      • -r： 递归复制，自动复制文件夹内部的文件夹，自动创建目标位置不存在的文件夹。多用于目录的复制
      • -p： 连同属性一起复制，而不是使用默认的属性
      • -d： 如果src-file属于link file，则复制为link file，而非文件本身
      • -a： 用于文件备份，等同于-prd
      • -i： 如果目标文件在复制之前已经存在，则在覆盖之前提示是否覆盖
      • -u： 如果源文件比目标文件新，则覆盖，否则跳过
      • -f： 强制复制
      • -s： 复制为符号链接文件，即为快捷方式文件，俗称为软连接（用于创建快捷方式）
      • -l： 复制为硬连接文件
  • rm（移除文件或目录）
    • 语法：rm [-fir] 文件或者目录（可以有多个文件，中间使用空格隔开）
    • 选项与参数：
      • -f： –force 忽略不存在的文件，不会给出警告信息
      • -i： –interactive 互动模式，删除前会询问是否动作
      • -r： –recursive 递归删除,同时删除该目录下的所有目录层
      • -v： –verbose 详细显示进行的步骤
    • 注意：rmdir只能删除空的目录，而rm命令可以删除任意目录和文件

    • 实例：

  • mv(移动文件与目录，或者修改名称)
    • 语法：mv [option] src destina(可以有多个源文件，中间使用空格隔开)
    • 选项与参数：
      • -f： –force 如果目标文件已经存在，不会询问，直接覆盖
      • -i： 互动模式，如果存在会询问是否覆盖
      • -u： 如果目标文件已经存在，并且源文件较新时才会升级

    • 实例
      mv a d
      将a目录下的所有文件移动到d目录下

1.3. 三、文件内容查看

  • cat（concatenate files and print on the standard output）
    • 由第一行开始显示文件内容
    • 语法：cat [option] 文件
    • 选项
      • -A： 可以列出一些特殊字符而不是显示空白
      • -b： 列出行号，但是空白行不显示行号
      • -E： 将结尾的断行字节$显示出来
      • -n： 列出行号，包括空白行
      • -T： 将tab按键以^|显示出来
      • -v： 列出一些看不出来的特殊字符
  • tac（concatenate and print files in reverse）
    • 倒序输出，文件内容从最后一行开始显示
  • nl（number line of files）
    • 显示文件以及行号
    • 语法：nl [option] 文件
    • 选项
      • -b： 指定行号的形式
        • -b a：列出空行
        • -b t：不列出空行
      • -n： 列出行号表示的方法
      • -w： 行号栏占用的位数
  • more
    • 一页一页翻动
  • less
    • 一页一页翻动
  • head
    • 取出文件前几行
  • tail
    • 取出文件后几行
