Linux下的文件的压缩、打包和备份

gzip工具

概述

  • GNUzip

  • 用于解压和压缩.gz格式的压缩文件

命令简介

  • 语法：gzip [选项] 源文件
  • 选项
    • -l：list，列出压缩文件信息
    • -d：decompress，解压缩的参数。
    • -r：recursive，递归处理
    • -c：stdout，将压缩后的数据输出到屏幕上，不改变原始文件
    • -t：test，检查压缩文件的一致性，即是否有错误
    • -v：verbose，显示压缩或解压缩的每个文件的名称和压缩比
    • -#：#表示数字，代表压缩等级。-1最快，但是压缩比最差，-9最慢，但是压缩比最好，默认为-6

tar工具

概述

  • tape archive

  • 用于打包和拆解文件，文件后缀为.tar

命令简介

  • 语法：tar 参数 文件或目录
  • 参数
    • -A：新增文件到已存在的包文件
    • -v：显示指令执行过程
    • -p：保留原来的文件权限和属性
    • -f：指定文件，放在参数最后
    • -x：拆解文件
    • -j：通过bzip2指令压缩或者解压缩文件，文件名最好为*.tar.bz2
    • -z：通过gzip指令压缩或者解压缩文件，文件名最好为*.tar.gz
