1. Icarus Verilog的使用

1.1. 概述

  • Icarus Verilog是一个轻量、免费、开源的Verilog编译器，基于C++实现，可以用于Verilog文件的语法检查以及一些基本的的时序仿真
  • 包含3个工具
    • iverilog：用于编译verilog和VHDL文件，进行语法检查，生成可执行文件
    • vvp：根据可执行文件，生成仿真波形文件
    • gtkwave：用于打开仿真波形文件，图形化显示波形
  • Iverilog常用参数
    • -o：用于指定生成文件的名称，-o test，如果不指定默认为a.out
    • -y：用于指定包含文件夹，当top.v中代用其他.v模块时，如果找不到调用模块就使用此命令显式指定,-y path
    • -tvhdl：把verilog文件转换为VHDL文件，-tvhdl

      1.2. Verilog编译仿真实际过程

  • 开始仿真前要testbench文件产生测试信号，同时注意文件中必须要有以下编译器专用语句，否则无法生成vcd文件
  • 编译
    • iverilog -o wave haedreg_top.v
    • 注意此时编译的应该是testbench文件
  • 生成波形文件
    • vvp -n wave -lxt2
    • 运行之后会在当前目录下生产.vcd文件
  • 打开波形文件
    • gtkwave wave.vcd
  • 转换为VHDL文件
    • iverilog -tvhdl -o a.vhd hardreg_top.v
  • VHDL文件的编译
    • 在原编译命令的基础上添加-g2012即可
