常用系统任务

输出信息
  • 语法格式
    • 这两个函数和系统任务的作用都是用于输出信息，$isaplsy会在输出之后进行换行,$write则不会
    • 二者会将p2,pn按照p1给定的格式输出，p1称为格式控制
  • 输出格式控制
    • 是使用双引号引起来的字符串
    • 包含格式说明和普通字符
    • 格式说明
      • 以%开头
    • 普通字符
      • 原样输出的字符，特殊字符可以通过转换序列输出
    • 输出数据的显示宽度
      • 输出列表中数据的显示宽度总是为表达式的最大可能值的数据宽度，十进制时，结果前的0会被省略，但是其他进制不会省略
      • 举例
    • 不确定值或者高阻值的输出规则
      • 十进制

        所有位都为不定值，输出x
        所有位都为高阻值，输出z
        部分位为不定值，输出X
        部分位为高阻值，输出Z

      • 十六进制和八进制

        某一位数所对应的所有二进制位都为不定值，则该位输出x
        某一位数所对应的所有二进制位都为高阻值，则该位输出z
        某一位数所对应的部分二进制位为不定值，则该位输出X
        某一位数所对应的部分二进制位为高阻值，则该位输出Z

文件输出
  • Verilog的结果通常输出到标准输出文件verilog.log中，可以将verilog的输出重新定向到选择的文件
  • 打开文件
    • 语法：$fopen("文件名");,文件句柄 = $fopen("文件名");
    • 返回值：32位的多通道描述符，多通道描述符中只有一位被设置为1。通道号与多通道描述符中被设置为1的位相对应(0-30位可以设置，31位为保留位)。
    • 说明
      • 对$fopen的每一次调用依次打开一个新的通道
      • 通道0被标准输出占用，且一直开放，所以此操作的返回值从通道1开始。
    • 举例
  • 写文件
    • 指令：$fdisplay，$fmonitor
    • 语法
    • 说明
      • p1,p2,pn可以是变量、信号名或者带引号的字符串，即为将要写入文件的内容。
      • 文件描述符是一个多通道描述符，可以是一个文件句柄或者多个文件句柄按位的组合。
      • Verilog会把输出写到与文件描述符中值为1的位相关的所有文件中。
    • 举例
  • 关闭文件
    • 指令：$fclsoe
    • 语法
    • 说明
      • 文件一旦被关闭就不能再写入
      • 多通道描述符中额相应位置会被置为0
        显示层次
  • 显示级别的层此名称
  • 语法：$display("%m");
  • 说明
    • 当执行到此语句时，会输出当前所在级别的全层次路径名
  • 举例
    选通显示
  • 命令：$strobe
  • 语法：同$display
  • 说明
    • 此语句总是在同时刻其他赋值语句执行完之后再执行，确保在同一时钟沿赋值的其他语句在执行完毕之后才显示数据。$display任务则是随机的
  • 举例
    值变转储文件
  • 值变转储文件(VCD)是一个ASCII文件，包含仿真时间、范围与信号的定义以及仿真运行过程中信号值的变化等信息。
  • 设计中的所有信号或者选定的信号集合在仿真过程中都可以被写入VCD文件，后处理工具可以把VCD文件作为输入并把层次信息、信号值和信号波形显示出来。
  • 命令
    • $dumpvars：选择需要转储的模块实例或者模块实例信号，$dumpvars(n,name)，n表示层次等级，即为只转储name所指模块的下第n层信号，n=0表示转储模块下各个层的所有信号
    • $dumpfile：选择VCD文件名称，$dumpfile("a.vcd")
    • $dumpon：选择转储过程的起点，启动转储过程
    • $dumpoff：选择转储过程的终点，停止转储过程
    • $dumpall：选择生成检测点

      调试用系统任务

      $monitor
  • 作用：提供了监控和输出参数列表中的表达式或者变量值的可能
  • 语法格式
    • $monitor(p1,p2,……,pn);
      • 此时规则和$display一样。当参数列表中的变量或者表达式的值发生变化的时候，整个参数列表中的变量或者表达式的值都将显示，如果同一时刻多个参数值发生变化，则只显示一次
      • 参数可以是$time系统函数，此时参数列表中的变量或者表达式的值同时发生变化的时刻可以通过标明同一时刻的多行输出来显示。$monitor($time,,"rxd = %b txd = %b",rxd,txd);,,,表示空参数，输出为空格
      • $display中也可以使用上述用法
    • $monitor;
    • $monitoron;：用于控制$monitor的启动和停止，此命令用于打开监控标志，启动监控任务，调用次任务时总是会输出当前参数列表的值，用于设置初始值
    • $monitoroff;：同上，用于关闭监控标志，停止监控任务
  • 注意
    • 任何时刻只能由一个$monitor起作用，并且常在initial中调用
      $time
  • 时间度量系统函数，可以得到当前的仿真时刻，有两种：$time和$realtime
  • $time
    • 返回一个64位的整数表示当前仿真时刻值，该时刻是以模块的仿真时间尺度为基准。
    • 实例
    • 说明
      • $time显示时刻受时间尺度比例的影响，输出时刻总是时间尺度的倍数(不一定是整倍数)
      • $time总是输出整数，如果不是整数则会自动取整再输出
  • $realtime
    • 和$time作用一致，但是返回的时间数字是一个实型数，也是以时间尺度为基准
    • 实例
      $finish
  • 语法格式
    • $finish;：默认参数值为1
    • $finish(n);
  • 作用
    • 退出仿真器，返回主操作系统，结束仿真过程
  • 不同参数值系统输出的特征信息
    • 0：不输出任何信息
    • 1：输出当前仿真时刻和位置
    • 2：输出当前仿真时刻、位置和在仿真过程中所用的memory以及CPU时间统计
      $stop
  • 语法格式
    • $stop;
    • $stop(n);
  • 作用
    • 设置EDA工具为暂停模式，在仿真环境下给出一个交互式的命令提示符，将控制权交给用户
  • 参数值同上
    $readmemb 和 $readmemh
  • 二者并用来从文件中读取数据到存储器。这两个任务可以在仿真的任何时刻被执行使用
  • 语法格式
    • $readmemb("<数据文件名>",<存储器名>);
    • $readmemb("<数据文件名>",<存储器名>,<起始地址>);
    • $readmemb("<数据文件名>",<存储器名>,<起始地址>,<结束地址>);
    • $readmemh("<数据文件名>",<存储器名>);
    • $readmemh("<数据文件名>",<存储器名>,<起始地址>);
    • $readmemh("<数据文件名>",<存储器名>,<起始地址>,<结束地址>);
