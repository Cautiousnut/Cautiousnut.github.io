常用指令

跳转指令

  • B [cond] label

    • Branch

    • 跳转到lable指向的位置

    • 采用相对寻址，PC = PC + label，跳转范围-32M ~ +32M。

    • 共24位表示偏移量，但是偏移量后两位一定为0，所以实际为26位。

  • BL [cond] label

    • Branch and Link
    • 跳转的同时保存断点，常用于子程序调用。
    • 采用相对寻址.LR = PC - 4 ,PC = PC + label.此时LR存储本跳转指令的下一条指令地址,作为程序返回之后的执行起点.
    • BL机器指令格式同上

  • BX [cond] Rm

    • Branch and Exchange
    • 跳转的同时切换处理器状态
    • 如果Rm的0位为1,则将CPSR的T标志位置位,切换到Thumb模式;否则T标志位复位,切换到ARM模式
    • 采用相对寻址,PC = PC + Rm

  • BLX [cond] label|Rm

    • 跳转的同时保存断点并可进行模式切换

存储器访问指令

共有三种类型的存储器访问指令

  • 单寄存器传输指令,LDR,STR
  • 多寄存器传输指令,LDM,STM
  • 交换指令,SWP
LDR、STR指令

  • Load Register`storage Register`

  • 从内存地址中加载一个字到寄存器.

  • 语法

    语法及指令格式
    LDR\STR[cond] Rd,addressing_mod
    • Rd:目标寄存器,将取出的数据存入此寄存器
    • addressing_mod:
指令名称 操作 指令描述
LDR 将一个字装入寄存器 LDR Rn,基址变址寻址,将地址中存储一个字放入Rn寄存器
STR 将寄存器中的值存入内存单元 STR Rn,基址变址寻址,将Rn中的内容存入给定的地址中
LDRB/STRB 将一个字节装入寄存器/ LDRB Rn,基址变址寻址,将地址中的存储的一个字节放入Rn寄存器
STRH/STRH 将一个半字装入寄存器/
STM、LDM指令

  • Store Multiple，Load Multiple

  • 将多个寄存器内容存储到指定内存

  • 会增加中断的延迟，因为ARM不会打断正在执行的指令，而去响应中断。一般编译器可以通过设置一次传送的最大寄存器数目来限制此延迟。

  • 语法

    语法及机器指令格式
    STM/LDM[cond]<addressing_mode> <Rn>[!],<register>[^]

    • cond:决定指令是否执行的条件

    • addressing_mode:

      • IA:increment after,传送后地址加4.操作堆栈时等于EA,空递增堆栈
      • IB:increment before,传送前地址加4.操作堆栈时等于FA,满递增堆栈
      • DA:decrement after,传送后地址减4.操作堆栈时等于ED,空递减堆栈
      • DB:Decrement before,传送前地址减4.操作堆栈时等于FD,满递减堆栈

    • Rn:Rn的值作为操作的基地址

    • !:操作完成之后将新的地址写回Rn

    • register:用{}括起来的寄存器列表。寄存器之间用,隔开,连续的寄存器可用-简写。

      小号寄存器存入低地址,大号存储器存入高地址。register_list16位对应16个寄存器,相应位为1,则代表选中相应的寄存器。

    • ^:将当前模式下的SPSR写入CPSR

  • 注意

    • 在指令中有!,且Rn在寄存器列表中时,如果Rn不是其中编号最小的寄存器,则会发生错误,结果不可预测.
SWP指令

  • signal data swap，单数据交换指令

  • 是原子操作，可用于实现信号量和互斥

  • 语法

    语法及机器指令格式
    SWP{cond}{B}Rd,Rm,[Rn]
    • cond：决定指令是否执行的条件
    • B：决定是交换字还是字节
    • Rd、Rm、[Rn]：将[Rn]地址对应的数据加载到Rd，并将Rm的值加载到[Rn]中

数据处理指令

杂项指令

MSR指令

  • 向PSR寄存器中写入一个数据

  • 语法

    语法及机器指令格式
    MSR{cond}<psr>,Rm，向PSR中写入数据
    MSR{cond}<psrf>,Rm，仅向PSR中写入标志寄存器，位对应写入NZCV标志位，下同
    MSR{cond}<psrf>,<#expression>，仅向PSR中写入标志寄存器
    • cond：决定指令是否执行的条件
    • psr：CPSR或者SPSR
    • psrf：CPSR_flg或者SPSR_flg
    • Rm：数据来源寄存器
    • #expression：第二操作数的立即数形式，必须满足条件
MRS指令

  • 将PSR寄存器中的内存读入一个寄存器中

  • 语法

    语法及机器指令格式
    MRS{cond} Rd,<psr>
    • cond：决定指令是否执行的条件
    • Rd：目标寄存器，不能是R15
    • psr：CPSR或者SPSR
SWI指令

  • software interrupt

  • 响应软件中断并进入管理模式

  • 语法

    语法及机器指令格式
    SWI{cond}<expression>
    • cond：决定指令是否执行的条件
    • expression：Evaluated and placed in the comment field，会被忽略，但可以用于传递参数给处理程序
