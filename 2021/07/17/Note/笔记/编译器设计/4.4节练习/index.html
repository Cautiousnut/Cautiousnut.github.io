4.4节练习

4.4.1

为下面的每个文法设计一个预测分析器，并给出预测分析表。你可能先要对文法进行提取左公因子或者消除左递归的操作。

  • 练习4.2.2（1）中的文法
  • 练习4.2.2（2）中的文法
  • 练习4.2.2（3）中的文法
  • 练习4.2.2（4）中的文法
  • 练习4.2.2（5）中的文法
  • 练习4.2.2（7）中的文法

    解答

  1. S -> 0 S 1 | 0 1

    • 消除左递归

    • 提取左公因子

    • 构造预测分析表

      非终结符 FIRST集合 FOLLOW集合
      S {0} {$,1}
      A {0,1} {$,1}
      非终结符\终结符 0 1 $
      S S -> 0 A
      A A -> S 1 A -> 1

  2. S -> + S S | * S S | a

    • 消除左递归

    • 提取左公因子

    • 构造预测分析表

      非终结符 FIRST集合 FOLLOW集合
      S {+,*,a} {$,+,*,a}
      非终结符\终结符 + * a $
      S S -> + S S S -> * S S S -> a

  3. S -> S ( S ) S | ε

    • 消除左递归

    • 提取左公因子

    • 构造预测分析表

      非终结符 FIRST集合 FOLLOW集合
      A {ε,(} {$}
      S {(} {$,)}
      C {(,)} {$,)}
      D {(,ε} {$,)}
      B {(,ε} {$,)}
      E {),(} {$,)}
      非终结符\终结符 ( ) $
      A A -> S A -> ε
      S S -> ( C
      C C -> S ) D C -> ) D
      D D -> S B D -> B D -> B
      B B -> ( E B -> ε B -> ε
      E E -> S ) B E -> ) D

  4. S -> S + S | S S | ( S ) | S * | a

    • 消除左递归

    • 提取左公因子

    • 构造预测分析表

      非终结符 FIRST集合 FOLLOW集合
      S {a,(} {$,),+,a,(,*}
      A {+,a,(,*,ε} {$,),+,a,(,*}
      非终结符\终结符 a ( ) * + $
      S S -> a A S -> ( S ) A
      A A -> S A;A -> ε A -> S A;A -> ε A -> ε A -> * A;A -> ε A -> + S A;A -> ε A -> ε
      易得此文法不是LL（1）文法

  5. S -> ( L ) | a;L -> L , S | S

    • 消除左递归

    • 提取左公因子

    • 构造预测分析表

      非终结符 FIRST集合 FOLLOW集合
      S {a,(} {$}
      L {a,(} {)}
      A {,,ε} {)}
      非终结符\终结符 ( ) , a $
      S S -> ( L ) S -> a
      L L -> ( L ) A L -> a A
      A A -> ε A -> , L

  6. bexpr -> bexpr or bterm | bterm ; bterm -> bterm and bfactor | bfactor ; bfactor -> not bfactor | (bexpr) | true | false ;

    • 消除左递归

    • 提取左公因子

    • 构造预测分析表

      非终结符 FIRST集合 FOLLOW集合
      bexpr {not,(,true,false} {$}
      A {or,ε} {$,)}
      bterm {not,(,true,false} {$,or,)}
      B {and,ε} {$,or,)}
      bfactor {not,(,true,false} {$,and,or,)}
      非终结符\终结符 ( ) or not and true false $
      bexpr bexpr -> bterm A bexpr -> bterm A bexpr -> bterm A bexpr -> bterm A
      A A -> ε A -> or bterm A A -> ε
      bterm bterm -> bfactor B bterm -> bfactor B bterm -> bfactor B bterm -> bfactor B
      B B ->ε B -> ε B -> and bfactor B B -> ε
      bfator bfactor -> ( bterm A ) bfactor -> not bfactor bfactor -> true bfactor -> false

    S S -> a A
    A A -> S B A -> ε A -> ε A -> ε
    B B -> + A B ->* A

      4.4.5

      文法 S -> aSa | aa 生成了所有由 a 组成的长度为偶数的串。我们可以为这个文法设计一个带回溯的递归下降分析器。如果我们选择先用产生式 S -> aa 展开，那么我们只能识别串 aa。因此，任何合理的递归下降分析器将首先尝试 S -> aSa。
  1. ！ 说明这个递归下降分析器识别输入 aa，aaaa 和 aaaaaaaa，但识别不了 aaaaaa。
