4.6.1

描述下列文法的所有可行前缀

  1. 练习4.2.2-1的文法 S->0S1|01
  2. ！ 练习4.2.1的文法 S->SS+|SS*|a
  3. ！ 练习4.2.2-3的文法 S->S(S)S|ε

解答

  1. S -> 0 S 1 | 0 1

    • 构造增广文法

    • 构造LR(0)自动机

    • 所有可行前缀

  2. S -> SS+|SS*|a

    • 构造增广文法
    • 构造LR(0)自动机
    • 所有可行前缀

  3. S -> S(S)|ε

    • 构造增广文法
    • 构造LL(0)自动机
    • 所有可行前缀

      4.6.2

      为练习4.2.1中的（增广）文法构造SLR项集。计算这些项集的GOTO函数。给出这个函数的语法分析表。这个文法是SLR文法吗？

      解答

  • 构造增广文法
  • 构造LL(0)自动机
  • 文法编号以及FOLLOW集合
    非终结符 FIRST集合 FOLLOW集合
    S’ {a} {$}
    S {a} {$,+,*a}
  • 构造语法分析表
    状态 ACTION GOTO
    a + * $ S' S
    0 s2 1
    1 s2 acc 3
    2 r4 r4 r4 r4
    3 s2 s4 s5 3
    4 r2 r2 r2 r2
    5 r3 r3 r3 r3
  • 易得此文法是SLR文法

    4.6.3

    利用练习4.6.2得到的语法分析表，给出处理输入aa*a+时的各个动作。

    解答

栈 符号 输入 动作
(1) 0 aa*a+$ 移入
(2) 0 2 a a*a+$ 根据S -> a进行规约
(3) 0 1 S a*a+$ 移入
(4) 0 1 2 Sa *a+$ 根据S -> a进行规约
(5) 0 1 3 SS *a+$ 移入
(6) 0 1 3 5 SS* a+$ 根据S -> SS*进行规约
(7) 0 1 S a+$ 移入
(8) 0 1 2 Sa +$ 根据S -> a进行规约
(9) 0 1 3 SS +$ 移入
(10) 0 1 3 4 SS+ $ 根据S -> SS+进行规约
(11) 0 1 S $ 接受

    4.6.5

    说明下面的文法

    S->AaAb|BbBa
    A->ε
    B->ε

是LL(1)的，但不是SLR(1)的。

解答

  • 是LL(1)的

    • 消除ε产生式
    • 消除左递归、提取左公因子
    • 求FIRST和FOLLOW集合
      非终结符 FIRST集合 FOLLOW集合
      S {a,b} {$}
      易得FIRST(ab) = {a}和FIRST(ba) = {b}不相交，且二者都不包含ε
      所以此文法是LL(1)的

  • 不是SLR(1)

    • 构造增广文法
    • 构造LR(0)自动机

      4.6.6

      说明下面的文法

    S->SA|A
    A->a

是SLR(1)的，但不是LL(1)的

解答

  • 是SLR(1)的

    • 构造增广文法

    • 构造LR(0)自动机

    • 构造语法分析表

      非终结符 FIRST集合 FOLLOW集合
      S’ {a} {$}
      S {a} {a,$}
      A {a} {a,$}
      状态 a $ S’ S A
      0 s3 1 2
      1 s3 r1 4
      2 r3 r3
      3 r4 r4
      4 r2 r2
      综上可得，此文法是LSR(1)的

  • 不是LL(1)的

    • 消除左递归

    • 提取左公因子

    • 计算FIRST集合和FOLLOW集合

      非终结符 FIRST集合 FOLLOW集合
      S {a} {$}
      B {a,ε} {$}
      A {a} {a,$}

    • 构造预测分析表

      非终结符\输入符号 a $
      S S -> AB
      B B -> AB B -> ε
      A A -> a

      变化后的文法是LL(1)的,但是原文法有左递归，所以不是LL(1)的

      4.6.7!!

考虑按照下面的方式定义的文法族 G_n：

S -> A_i b_i         其中1<=i<=n
A_i-> a_j A_j | a_j    其中1<=i,j<=n 且i<>n

说明：

  1. G_n有 2n^2-n 个产生式
  2. G_n有 2^n+n^2+n 个 LR(0) 项集
  3. G_n是 SLR(1) 的

关于LR语法分析器的大小，这个分析结果说明了什么？

4.6.8!

我们说单个项可以看做一个 NFA 的状态，而有效项的集合就是一个 DFA 的状态。对于练习4.2.1的文法 S->SS+|SS*|a

  1. 根据“将项看作一个NFA的状态”部分中的规则，画出这个文法的有效的转换图（NFA）
  2. 将子集构造算法（算法3.20）应用于在（1）部分构造得到的NFA。得到的DFA和这个文法的LR(0)项集比有什么关系
  3. ！！ 说明在任何情况下，将子集构造算法应用于一个文法的有效项的NFA所得到的就是该文法的 LR(0) 项集
