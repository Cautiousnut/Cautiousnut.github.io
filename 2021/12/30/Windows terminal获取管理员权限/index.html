Windows terminal获取管理员权限

安装Scoop包管理工具

  • 以管理员身份进入powershell，并按照scoop文档进行安装即可https://github.com/ScoopInstaller/Scoop

  • 输入scoop help查看是否安装成功

安装sudo工具

  • 使用命令scoop install sudo --global安装

  • 之后使用指令时在前边加上sudo即可以管理员身份运行指令
