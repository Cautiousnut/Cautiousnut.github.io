ARP洪泛和欺骗

ARP洪泛攻击

产生原因

交换机或者路由器接收到ARP请求包时，如果没有相应的IP地址匹配，则会将ARP请求包在网络中进行广播，同样别的网络节点接收到此ARP请求之后也可能进行广播。在此过程中,如果链路中存在环路,则可能造成ARP包无休止的在链路中进行转发,由此造成网络效率下降甚至中断.

rprimary -> a | b

  5. 无左公因子

  8. 适用

为下面的每个文法设计一个预测分析器，并给出预测分析表。你可能先要对文法进行提取左公因子或者消除左递归的操作。

    解答

  1. S -> 0 S 1 | 0 1

    • 消除左递归

    • 提取左公因子

    • 构造预测分析表

      非终结符 FIRST集合 FOLLOW集合
      S {0} {$,1}
      A {0,1} {$,1}
      非终结符\终结符 0 1 $
      S S -> 0 A
      A A -> S 1 A -> 1

  2. S -> + S S | * S S | a

    • 消除左递归

    • 提取左公因子

    • 构造预测分析表

      非终结符 FIRST集合 FOLLOW集合
      S {+,*,a} {$,+,*,a}
      非终结符\终结符 + * a $

  3. S -> S ( S ) S | ε

    • 消除左递归

    • 提取左公因子

    • 构造预测分析表

      S {(} {$,)}
      C {(,)} {$,)}
      D {(,ε} {$,)}
      B {(,ε} {$,)}
      E {),(} {$,)}
      非终结符\终结符 ( ) $

      S S -> ( C
      C C -> S ) D C -> ) D
      D D -> S B D -> B D -> B
      B B -> ( E B -> ε B -> ε
      E E -> S ) B E -> ) D

  4. S -> S + S | S S | ( S ) | S * | a

    • 消除左递归

    • 提取左公因子

    • 构造预测分析表

      非终结符 FIRST集合 FOLLOW集合
      S {a,(} {$,),+,a,(,*}
      A {+,a,(,*,ε} {$,),+,a,(,*}

      S S -> a A S -> ( S ) A
      A A -> S A;A -> ε A -> S A;A -> ε A -> ε A -> * A;A -> ε A -> + S A;A -> ε A -> ε

    • 消除左递归

    • 提取左公因子

    • 构造预测分析表

      L {a,(} {)}
      A {,,ε} {)}

      L L -> ( L ) A L -> a A

  6. bexpr -> bexpr or bterm | bterm ; bterm -> bterm and bfactor | bfactor ; bfactor -> not bfactor | (bexpr) | true | false ;

    • 提取左公因子

      bterm {not,(,true,false} {$,or,)}
      B {and,ε} {$,or,)}
      bfactor {not,(,true,false} {$,and,or,)}
      非终结符\终结符 ( ) or not and true false $
      bexpr bexpr -> bterm A bexpr -> bterm A bexpr -> bterm A bexpr -> bterm A
      A A -> ε A -> or bterm A A -> ε
      bterm bterm -> bfactor B bterm -> bfactor B bterm -> bfactor B bterm -> bfactor B
      B B ->ε B -> ε B -> and bfactor B B -> ε
      bfator bfactor -> ( bterm A ) bfactor -> not bfactor bfactor -> true bfactor -> false
